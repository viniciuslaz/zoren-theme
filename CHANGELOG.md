## [1.3.0] - 2025-02-03

### Added

- **Zoren Breeze theme** - A clean, eye-friendly dark theme featuring:
  - Soft blue-gray base (#1e2228) instead of pure black for reduced eye strain
  - Carefully balanced contrast ratios optimized for long coding sessions
  - Natural color palette with moderate saturation to prevent visual fatigue
  - Warm syntax highlighting with green strings, golden yellows, and soft purples
  - Smooth gradual transitions between UI elements
  - Enhanced readability with thoughtful use of blue and cyan accents
